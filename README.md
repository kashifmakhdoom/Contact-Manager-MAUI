## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

List of things you need to use and how to install them:

- Visual Studio 2022 | Visual Studio Code
- MAUI Workload

### Installation

Setup:

1. dotnet workload install maui, maui.android, maui.ios
2. Clone the repo:
   ```sh
   git clone https://github.com/kashifmakhdoom/Contact-Manager-MAUI.git
   ```
3. Install Nuget packages:
   ```sh
   dotnet restore
   ```
4. dotnet run | execute from Visual Studio
